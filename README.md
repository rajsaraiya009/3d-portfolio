# Raj Saraiya - 3D AI Specialist Portfolio

An interactive, premium 3D portfolio website showcasing the expertise of Raj Saraiya, a Senior Data Scientist and AI Strategic Leader. Built with React, Three.js, and GSAP for a high-performance, immersive user experience.

![Portfolio Preview](https://github.com/rajsaraiya009/3d-portfolio/raw/main/public/preview.png) *(Note: Add a real preview image path if available)*

## 🚀 Experience the Magic

This portfolio isn't just a resume; it's a 3D journey through:
- **Interactive 3D Character**: A custom-animated character that reacts to your movement and scrolling.
- **AI-Driven Visuals**: Dynamic lighting system with high-precision soft shadows and environment mapping.
- **Ultra-Smooth Scrolling**: Powered by GSAP ScrollSmoother for a buttery-smooth, cinematic feel.
- **Strategic Data Storytelling**: Tailored sections for Data Science & AI, Data Analytics, and professional career history.

## 🛠 Tech Stack

- **Core**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **3D Engine**: [Three.js](https://threejs.org/) with [@react-three/fiber](https://github.com/pmndrs/react-three-fiber) & [@react-three/drei](https://github.com/pmndrs/drei)
- **Animation**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, ScrollSmoother, SplitText)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Typography**: Inter (Modern sans-serif)

## 🏃‍♂️ Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajsaraiya009/3d-portfolio.git
   cd 3d-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production Build

To create a production-ready bundle:
```bash
npm run build
```
The output will be in the `dist` folder.

## 📁 Project Structure

- `src/components/Character`: 3D Scene, Lighting, and Character models.
- `src/components`: UI components (About, Career, Work, WhatIDo).
- `src/components/utils`: Animation and Scroll logic (GSAP targets).
- `public/models`: 3D assets and environment maps.
